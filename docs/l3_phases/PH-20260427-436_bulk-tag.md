# PH-436: 一括タグ付け bulk-edit UI (Nielsen H7)

## 問題

use-case-friction-v2 case 2 (同人ライブラリ) H7 severity 3:
1 件ずつ右パネルでタグ付けは batch 100 件で苦行。

## 改修

### Library 複数選択モード

`LibraryMainArea.svelte`:

- 複数選択モードトグル (チェックボックス UI)
- 選択中アイテム ID を `Set<string>` で保持
- LibraryCard / LibraryRow に選択状態 prop 追加 + クリックで toggle
- Shift+click で範囲選択

### action bar

選択 1 件以上で表示:

- 「タグ付け (N 件)」 → タグ選択 popover → 一括 add/remove
- 「削除 (N 件)」 → 確認 dialog → 一括 delete
- 「キャンセル」 → 選択クリア

### Rust 側 IPC

- `cmd_bulk_add_tag(item_ids: Vec<String>, tag_id: String)` — transaction batch INSERT
- `cmd_bulk_remove_tag(item_ids: Vec<String>, tag_id: String)`
- `cmd_bulk_delete_items(item_ids: Vec<String>)` — transaction batch DELETE

## 受け入れ条件 (batch-96 minimal scope)
