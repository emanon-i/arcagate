# PH-20260226-002: ランチャーMVP（M1相当）

## 目的

毎日使えるランチャーのコア体験を確立する。「ホットキー → 名前入力 → Enter → 起動」の最速経路を実現し、ゲーム・開発ツール・スクリプトを統一アイテムモデルで管理する。

## 機能詳細

### F-20260226-005: アイテム登録・管理

任意の実行ファイル・URL・フォルダ・スクリプトを統一アイテムモデルで登録・編集・削除できる。

**技術要素**:

- L2の `items` テーブル（5種類: exe / url / folder / script / command）
- `categories`, `tags`, `item_categories`, `item_tags` テーブル
- ItemService + ItemRepo（L2 §2.3 Service Layer設計）
- IPC: `item_create`, `item_update`, `item_delete`, `item_get`, `category_list`, `category_create`, `tag_list`, `tag_create`
- Tauri plugin: `tauri-plugin-dialog`（ファイル選択）, `tauri-plugin-fs`（D&D）
- アイコン自動抽出: `src-tauri/src/utils/icon.rs`

**受け入れ条件**:

- [ ] exe / URL / フォルダ / スクリプト / コマンドの5種類をUIから登録できる
- [ ] ラベル・アイコン・カテゴリ・検索用別名（aliases）を設定できる
- [ ] exeファイルのドラッグ&ドロップで即登録でき、アイコンが自動取得される
- [ ] 登録済みアイテムの編集・削除ができる
- [ ] カテゴリ・タグの作成と、アイテムへの紐付けができる

### F-20260226-006: コマンドパレット検索・起動

コマンドパレットから名前でアイテムを検索・選択・起動できる。Arcagateのコアインターフェース。

**技術要素**:

- shadcn-svelte `Command` コンポーネント（cmdkパターン）
- `src/lib/components/palette/` (Palette.svelte, PaletteItem.svelte, PaletteInput.svelte)
- `src/lib/state/palette.svelte.ts`（PaletteState クラス、$state/$derived）
- IPC: `item_search`, `item_launch`
- LaunchService + LauncherモジュールL2 §2.2: `src-tauri/src/launcher/`（exe/url/script/folder/command別）
- `tauri-plugin-shell`（外部プロセス起動）

**受け入れ条件**:

- [ ] コマンドパレットにテキストを入力すると、ラベル・aliases でリアルタイムに絞り込まれる
- [ ] 絞り込み結果をキーボード（↑↓）で選択し、Enterで起動できる
- [ ] exe / URL / フォルダ / スクリプト / コマンドの各5タイプが正しく起動される（exe→プロセス起動、URL→ブラウザ、フォルダ→エクスプローラー、スクリプト→指定シェル、コマンド→内部コマンド実行）
- [ ] 検索結果にアイコン・カテゴリが表示される

### F-20260226-007: システムトレイ常駐・ホットキー

システムトレイに常駐し、グローバルホットキーでどの画面からでもコマンドパレットを呼び出せる。

**技術要素**:

- Tauri tray-icon（built-in feature）
- `tauri-plugin-global-shortcut`
- `tauri-plugin-autostart`
- IPC event: `hotkey-triggered`
- ConfigService（ホットキー設定の永続化）

**受け入れ条件**:

- [ ] アプリがシステムトレイに常駐し、トレイアイコンが表示される
- [ ] トレイアイコンの右クリックメニューから「表示」「設定」「終了」が選択できる
- [ ] グローバルホットキー（デフォルト設定）でコマンドパレットが表示/非表示をトグルできる
- [ ] Windows起動時の自動起動をON/OFFできる
- [ ] ホットキーのキーバインドを設定画面から変更できる

### F-20260226-008: 起動ログ・ソート

起動回数・最終起動日時を記録し、頻度順・最近使った順でアイテムを表示できる。

**技術要素**:

- `launch_log` テーブル + `item_stats` テーブル（L2 §4）
- LogService + LogRepo
- IPC: `log_recent`, `log_frequent`
- LaunchService 内で起動時に自動記録

**受け入れ条件**:

- [ ] アイテム起動時に `launch_log` に自動的にログが記録される
- [ ] ログ記録と統計更新が単一トランザクションで実行され、部分失敗が発生しない
- [ ] `item_stats` の起動回数・最終起動日時が更新される
- [ ] コマンドパレットで「最近使った順」「よく使う順」のソートが選択できる
- [ ] デフォルトのソート順を設定画面から変更できる

### F-20260226-009: データエクスポート/インポート

DB・アイコン・設定を含む全データアーカイブをエクスポート/インポートできる。バックアップ・移行用途。

**技術要素**:

- IPC: `data_export`, `data_import`
- `tauri-plugin-dialog`（保存先/読み込み元の選択）
- SQLite Backup API (`sqlite3_backup_init`) でWAL安全コピー
- icons/ ディレクトリを含む
- zip アーカイブ形式（`.arcagate-backup.zip`）
- アプリ側でチェックポイント後にバックアップ実行

**受け入れ条件**:

- [ ] 設定画面からDB + アイコン + 設定を含むzipアーカイブとしてエクスポートできる
- [ ] SQLite Backup APIを使用し、WALモード稼働中でもデータ整合性が保たれる
- [ ] エクスポートしたzipからインポートしてDB・アイコン・設定を完全復元できる
- [ ] インポート時に既存データの上書き確認ダイアログが表示される

### F-20260226-010: セットアップウィザード

初回起動時のシンプルなセットアップウィザード。最低限の設定を案内する。

**技術要素**:

- `src/lib/components/setup/`（モーダルダイアログとして表示）
- IPC: `setup_get_status`, `setup_complete`
- SetupService
- ConfigService（セットアップ完了フラグの永続化）

**受け入れ条件**:

- [ ] 初回起動時にセットアップウィザードがモーダルダイアログとして表示される
- [ ] ホットキーの確認・変更、自動起動の設定ができる
- [ ] ウィザード完了後は再表示されない（config テーブルで管理）
- [ ] 設定画面から「セットアップをやり直す」で再実行できる

### F-20260226-011: センシティブコンテンツの隠蔽

タグに「デフォルト非表示」フラグを設定し、ホットキーまたはパスワードで表示/非表示をトグルする。

**技術要素**:

- `tags.is_hidden` フラグ（L2 §4）
- IPC: `visibility_toggle`, `tag_update`（既存タグの `is_hidden` フラグ変更用）
- argon2 パスワードハッシュ（L2 §2.9）
- セッション中のメモリ上トグル状態管理

**受け入れ条件**:

- [ ] タグ作成・編集時に「デフォルト非表示」フラグを設定できる
- [ ] 非表示フラグ付きタグに紐付くアイテムがデフォルトで検索結果・一覧から除外される
- [ ] ホットキーまたはパスワード入力で表示/非表示をトグルできる
- [ ] パスワードは argon2 でハッシュ化して config テーブルに保存される
- [ ] パスワード未設定の場合はホットキーのみでトグル可能

## Exit Criteria

L0 M1成功条件に準拠:

- ゲーム起動・Blender複数バージョン切り替え・Claude Code起動の3ユースケースをノーエラーで実行できる
- ホットキー → 名前入力 → Enter → 起動のフローを10回繰り返してエラーが発生しない
- 常駐時メモリ: タスクマネージャーのWorking Set値で計測。アイテム50件登録・Idle 1分後に100MB以下
- 起動レイテンシ: AutoHotKey等でホットキー押下〜WebView描画完了を計測。連続20回中P95が2秒以内
- バイナリサイズ: `pnpm tauri build` 出力の `.exe` ファイルサイズが20MB以下
