# PH-003-B: カテゴリプレフィックス + 内蔵コマンド

**対応REQ**: REQ-20260226-008
**元機能**: F-20260226-012 (rest)
**前提**: PH-003-A 完了

コマンドパレットのカテゴリプレフィックス絞り込みと、内蔵コマンド（電卓・クリップボード履歴）を実装する。

## 技術要素

- `categories.prefix` フィールド（L2 §4: M2a用）— DBカラムが存在しない場合はマイグレーション追加
- GUI: コマンドパレットの検索ボックスでプレフィックス（例: `gm:`）を検出してカテゴリ絞り込み
- 内蔵コマンドプロバイダー: `CommandProvider` trait を実装した `BuiltinCommandProvider`
- 電卓: `meval` または手実装の簡易数式パーサー（`+`, `-`, `*`, `/`, `()` のみ対応で可）
- クリップボード履歴: `tauri-plugin-clipboard-manager` または Rust 側のクリップボード取得

## 受け入れ条件

- [ ] コマンドパレットで `gm:` のようにカテゴリプレフィックスを入力するとそのカテゴリのアイテムのみに絞り込まれる
- [ ] カテゴリに `prefix` を設定する UI が存在する（設定画面またはカテゴリ編集フォーム）
- [ ] コマンドパレットで `= 1+2*3` などの数式を入力すると計算結果が表示される
- [ ] コマンドパレットでクリップボード履歴を検索・選択して貼り付けできる
- [ ] `pnpm verify` が全通過する
