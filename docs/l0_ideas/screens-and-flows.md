# Arcagate 画面とユーザーフロー

## 主要画面

### Palette（コマンドパレット）

最速経路。どの画面にいてもホットキーで呼び出し、名前を打って Enter で起動する。

**こうあってほしい:**

- ホットキー押下から表示まで 1 秒以内
- 名前の一部を打つだけで登録しているアイテムのラベル名のファジーマッチで候補が出る
- キーボードだけで完結する（↑↓ + Enter）
- 閉じたいときは Esc またはコマンドパレット以外をクリック

**典型的な使い方:**

- ゲームを起動したい → Ctrl+Shift+Space → ゲーム名の一部を打つ → Enter
- Blender の特定バージョンを起動 → `blen4` と打つ → Enter
- Claude Code を起動 → `claude` と打つ → Enter

---

### Library（ライブラリ）

全登録アイテムの一覧。アイテムを登録・タグ付け・起動する。

**こうあってほしい:**

- exe のドラッグ&ドロップで即登録（アイコン自動取得）
- Webサイトもドラックアンドドロップで追加できる（ページ名自動取得）
- ブックマークのファイルを取り込んで一部選択または一括選択などでアイテムとして取り込める（取り込み時にタグも付与できる）
- タグで分類・絞り込みができる
- 起動頻度・最終起動日時順に並び替えられる
- カードサイズ S/M/L を選べる（ゲームのサムネイルを大きく見たい、ツールはコンパクトにしたい）
- アイテム詳細でメタデータ・タグ・アイコン・表示設定を編集できる
- タグにはURL、EXEやフォルダなど形式によるタグとworkspace名のタグのシステムタグをつける
- ユーザーが付けるタグはユーザータグでデフォルトは何もなし
- 不要なアイテムを非表示設定で隠せる

**典型的な使い方:**

- 同人ゲームを多数登録し、タグ「同人 RPG」で絞り込んでサムネイル一覧で選ぶ
- Library カードをクリック/ダブルクリックで起動

---

### Workspace（ワークスペース）

ウィジェットを自由配置するホーム画面。複数ページを作ってシナリオ別に使う。

**こうあってほしい:**

- ページごとにウィジェットを配置して保存される
- よく使うものをワンクリックで起動できる（Favorites / 特定の Item ウィジェット）
- プロジェクトフォルダの一覧と Git ブランチ状態を確認できる（Projects ウィジェット）
- 監視フォルダに追加された exe が自動でリストに現れる（ExeFolder ウィジェット）
- 監視フォルダに追加されたbat,ps1,shellなどのスクリプトが自動でリストに現れる
- 画像をドラックアンドドロップで配置できてスクラップできる（ウィジェットとして画像表示し、アイテムにも登録）
- textファイルをドラックアンドドロップで配置できてファイル名や文字数、サイズ、更新作成時間などのメタデータがわかり、中身を見れる（Markdown形式ならYAMLフロントマターも表示）（ウィジェットとしてメタデータとファイルの中身を表示してアイテムにも登録）
- （おまけ）メモを即座に書けて、開くたびに残っている（QuickNote ウィジェット）
- （おまけ）チェックリストで日次タスクを管理できる（DailyTask ウィジェット）
- ウィジェットでworkspaceで追加されたアイテムはworkspace名のタグをシステムタグとしてつける

**ページ構成の例:**

- Gaming ページ: Favorites（ゲーム）+ Recent + Stats
- Dev ページ: Projects + Favorites（IDE, ターミナル）+ ClipboardHistory + QuickNote
- Daily ページ: DailyTask + SystemMonitor + Snippet

**ウィジェット操作:**

- ウィジェットはドラッグで移動、8 方向リサイズ可能
- Ctrl+Z / Ctrl+Y で Undo/Redo
- ウィジェット同士は重ならない（重ならない位置に自動探索、見つからなければ追加しない）
- 無限にCanvasが広がっておりどこでも配置できるが軽量
- 全部のウィジェットまたは複数選択したウィジェットを画面いっぱいに収める機能

### Settings（設定）

アプリ全体の設定。2 ペイン構成（左側カテゴリ選択、右側内容）。

**カテゴリ:**

- **General**: ホットキー、自動起動
- **Library**: カードサイズ、デフォルトソート
- **Appearance**: テーマ選択・カスタマイズ
- **Data**: エクスポート/インポート、DB バックアップ
- **About**: バージョン、アップデート確認

**こうあってほしい:**

- 設定を変えたら即反映（保存ボタンなし）
- テーマは選んだ瞬間にプレビューで見た目が変わる
- ホットキーは入力欄にキーを押すだけで登録できる

---

### Onboarding（セットアップウィザード）

初回起動時のみ。シンプルに。

- ホットキーの確認（Ctrl+Shift+Space が既定）
- Windows 起動時に自動起動するかどうか
- 完了 → Library を開く

---

## 主要なユーザーフロー

### ゲームを起動する（最速経路）

1. Ctrl+Shift+Space → Palette を開く
2. ゲーム名の一部を打つ
3. Enter → 起動

### ゲームを登録する

1. Library を開く
2. exe ファイルをドラッグ&ドロップ
3. ラベル・タグを設定して保存

### 作業を始める

1. Workspace の Dev ページを開く（タブクリック）
2. Projects ウィジェットでプロジェクトを選択
3. Favorites からエディタ・ターミナルを起動

または

1. コマンドパレットでworkspace名のdevと入力
2. dev 入力後に予測でworkspaceに登録されているアイテムが出てくる
3. 街灯のアイテムを上下キーで選択した後エンターで事前に設定またはデフォルトの起動方法で起動する
