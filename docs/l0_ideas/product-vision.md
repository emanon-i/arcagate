# Arcagate — 製品ビジョン

## 何のためのアプリか

PC 上に散在するゲーム・開発ツール・スクリプト・URL を一箇所に登録し、即座に起動できる個人用ランチャー。Steam・DMM ゲーム・ブラウザゲーム・各種ランチャーをいちいち開かず、Arcagate だけ開けば全部にアクセスできる。

毎日開くツールとして常駐させる。配布・販売に耐える品質を常に狙う（GitHub public、将来的な配布・販売はオプションとして開いている）。

## 誰のためか

自分自身（開発者）。ゲームも開発ツールも日常的に多数使い分けており、起動までの手数を最小化したい。キーボード操作を好む。

## 何が特徴か

他の既存ランチャー（Raycast / Flow Launcher / Listary / Playnite）はゲーム特化か開発ツール特化かのどちらかで、両方を同一モデルで扱えない。Arcagate は `.exe / URL / フォルダ / スクリプト` を全て同一アイテムとして扱う。

## 現在存在する画面

### Library（ライブラリ）

登録されたアイテムの一覧。カード表示で、サムネイル・タグ・起動ログを管理する。タグによるフィルタ、頻度/最近使った順のソート、詳細パネルからの編集ができる。カードサイズは S/M/L から選択可能。

### Workspace（ワークスペース）

ページ + ウィジェットで構成されるカスタマイズ可能なホーム画面。よく使うもの・最近使ったもの・プロジェクト一覧・監視フォルダ・タスク・メモ・クリップボード履歴などのウィジェットを自由配置する。ページは複数作成でき、シナリオ別のランチャーパッドとして使う（例: Gaming ページ、Dev ページ）。

### Palette（コマンドパレット）

グローバルホットキー（Ctrl+Shift+Space）で表示されるフローティングウィンドウ。名前を打ってEnterで起動するのが最速経路。システムトレイ常駐なので、どの画面からでも呼び出せる。

### Settings（設定）

ホットキー、自動起動、テーマ、Library カードサイズ、監視フォルダ、データエクスポート/インポート、アップデートなどを管理する 2 ペイン構成の設定画面。

### Onboarding（セットアップウィザード）

初回起動時のみ表示。ホットキー設定と自動起動の有無を確認して、すぐ使える状態にする。

## 現在実装済みのウィジェット（12種）

- **Favorites** — お気に入りアイテム固定表示
- **Recent** — 最近起動したアイテム
- **Projects** — 監視フォルダ内プロジェクト一覧（Git ステータス表示）
- **Item** — 特定アイテムのピン留め
- **Stats** — 起動統計
- **QuickNote** — 自動保存メモ
- **ExeFolder** — フォルダ内の exe 一覧
- **DailyTask** — 日次チェックリスト
- **Snippet** — テキストスニペット
- **ClipboardHistory** — クリップボード履歴
- **FileSearch** — ファイル検索
- **SystemMonitor** — CPU / メモリ表示

## スコープ外（今後も実装しない）

- クラウド同期
- Linux / macOS ネイティブ
- ターミナルエミュレータ統合
- ファイルマネージャー / セマンティック検索
- コンテキストメニュー統合（Shell Extension）
